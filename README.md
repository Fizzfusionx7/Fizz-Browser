# 🌐 Fizz Browser

A fast, lightweight and customizable browser powered by Chromium.

---

## 🚀 Download

Download the latest version from the Releases page:

- Open **Releases**
- Download `main.exe`
- Start the EXE

---

## 📦 Important

The `browser_data` folder is created automatically on the first launch because it contains many files and is too large to include in the repository.

---

## ✨ Features

- Chromium-based browser
- Lightweight UI
- Custom browser data folder
- Fast startup
- Open-source
